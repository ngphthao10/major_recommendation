## Setup
### Yêu cầu
- Python 3.11.9

### Cài đặt
```bash
$ pip install -r client_app/requirements.txt
$ pip install -r webservice/requirements.txt
```

## Chạy
### web service
```bash
$ cd frontend
$ npm i
$ npm start
```

### client app
```bash
$ cd backend
$ pip install -r requirements.txt
$ python app.py
```
