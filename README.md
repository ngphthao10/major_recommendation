## Setup
### Yêu cầu
- Python 3.11.9
- Node.js (link download: https://nodejs.org/en) 

## Chạy

```bash
$ git clone <repo_url>
```

### Frontend
```bash
$ cd frontend
$ npm i
$ npm start
```

### Backend
```bash
$ cd backend
$ pip install -r requirements.txt
$ python app.py
```
